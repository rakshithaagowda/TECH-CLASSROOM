# 🎓 Tech Classroom

Tech Classroom is a web-based learning platform for technical subjects that provides a structured and interactive learning experience. It includes quizzes, classroom management features, a student dashboard, to-do tasks, chatbot support, and a resource library.

---

## 🌟 Overview

Tech Classroom is built to simulate a digital classroom environment where students can:
- Learn technical concepts
- Attempt quizzes at different levels
- Track their performance
- Access learning resources
- Manage daily tasks

---

## 🚀 Features

- 🏫 Classroom-based learning system  
- 🧠 Level-based technical quizzes  
- 📊 Student performance tracking  
- 📚 Resource library for study materials  
- 🤖 Basic chatbot support for quiz
- ✅ To-do / task management system  

---

## 🛠️ Tech Stack

- Backend: Python (Flask)  
- Frontend: HTML, CSS, JavaScript  
- Database: SQLite

▶️ How to Run Locally
1. Clone the repository
git clone https://github.com/rakshithaagowda/TECH-CLASSROOM.git
2. Navigate into the project folder
cd TECH_CLASSROOM/backend
3. Create a virtual environment
python -m venv venv
4. Activate the virtual environment
Windows:  venv\Scripts\activate
Mac/Linux:  source venv/bin/activate
5. Install dependencies
pip install -r requirements.txt
6. Run the backend server
python app.py
7. Open the frontend
Go to the frontend folder and open:
frontend/index.html
(You can double-click it or open with Live Server if using VS Code)
